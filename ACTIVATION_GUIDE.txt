MARKETMIND AI V5 — GITHUB UPDATE

1. Unzip MarketMind_AI_v5.zip.
2. Open the existing marketmind-ai GitHub repository.
3. Tap Add file > Upload files.
4. Upload index.html, manifest.json, service-worker.js, config.json, README.md and the icons folder.
5. Replace the old files and commit to main.
6. Wait 1–3 minutes.
7. Open the existing GitHub Pages URL in Safari and refresh.
8. If an older version remains, remove the Home Screen app and add it again.

V5 adds a Results Calendar (Q1 FY27 dates, your holdings flagged) and a My Portfolio
section: add your holdings and see, for each story, what % of your portfolio (by
value) is actually exposed — not just which stock prices move. Holdings are saved
in this browser's local storage only; they are not synced or backed up elsewhere.
